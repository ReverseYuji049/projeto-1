# Godot Studies

This repository contains my studies of Godot based on the YouTube course by Professor Rafael Forbeck.

## About

This project is part of my learning journey with the **Godot Engine**, where I explore core concepts of 2D game development such as:

* Player movement and physics
* State machines (idle, walk, jump, etc.)
* Animation handling
* Input system
* Basic game mechanics

The goal is to build a solid foundation for creating more complex games in the future.

## Technologies

* [Godot Engine 4.x](https://godotengine.org/)
* GDScript

## Purpose

This repository is intended for:

* Practicing game development concepts
* Tracking my progress over time
* Serving as a reference for future projects

## Notes

The code may change frequently as I continue learning and improving my skills.
